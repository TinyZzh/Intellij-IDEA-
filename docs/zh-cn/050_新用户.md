# 1. 新用户推荐设置和参数

本章节主要针对首次或刚开始准备使用Intellij IDEA的用户。提供一些列推荐设置或参数。

## 1.1. 基础设置 - 默认文件编码格式

[`Editor` -> `File Encodings`]Win 10系统默认的文件编码为`GBK`。IDE默认编码和系统一致。推荐修改为`UTF-8`编码格式，能有效的避免乱码。
